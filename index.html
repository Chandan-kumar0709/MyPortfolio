Wildlife Diversity & Conservation Portal (MCA Capstone)

A multi‑page, fully responsive portal highlighting India's wildlife diversity with interactive exploration and student‑friendly learning tracks. Built with Tailwind + Vanilla JS (optional Node.js backend) and a JSON‑first content system for species, states, and conservation projects.

Key Modules

  • Species Profiles (Tiger, Elephant, etc.) with habitat, diet, IUCN status.
  • State‑wise Presence: filterable views & basic map overlays.
  • Conservation Projects: Govt/NGO initiatives with milestones.
  • Media Gallery: images, infographics & short explainers.

Tech & Architecture

  • Frontend: HTML + Tailwind + Vanilla JS (no heavy frameworks).
  • Routing: multi‑page with shared components.
  • Storage: JSON schemas for species.json, states.json, projects.json.
  • Performance: lazy images, code splitting, minified assets.
  • Accessibility: keyboard nav, labeled controls, color contrast.

Sample Data Model

{
  "species": [
    {
      "id": "tiger",
      "name": "Bengal Tiger",
      "habitat": ["grasslands","tropical forests"],
      "states": ["MP","MH","WB","Assam"],
      "status": "Endangered",
      "projects": ["Project Tiger"]
    }
  ],
  "projects": [
    { "id": "proj-tiger", "name": "Project Tiger", "year": 1973, "type": "Govt" }
  ]
}

UX Highlights

Global search with auto‑suggest, state/habitat filters, animated cards on scroll, and a guided learning path for students. Focus on clarity, discoverability, and performance.
